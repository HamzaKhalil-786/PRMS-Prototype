# PRMS Prototype (Patient Record Management System)

Overview

PRMS (Patient Record Management System) is a prototype web-based application designed to manage hospital patient records efficiently. It provides role-based access for administrators and operators.

 Database Setup

1. Open **phpMyAdmin** from XAMPP.
2. Create a new database:

   ```
   hospital_db
   ```
3. Import the provided database file:

   ```
   hospital_db.sql
   ```

---

Project Installation

1. Copy the project folder:

   ```
   Cs619_prototype_PRMS
   ```
2. Paste it into the XAMPP `htdocs` directory:

   ```
   C:\xampp\htdocs\
   ```

---

Running the Application

1. Start **Apache** and **MySQL** using XAMPP Control Panel.
2. Open your web browser and navigate to:

   ```
   http://localhost/Cs619_prototype_PRMS
   ```

---

Requirements

* XAMPP (Apache & MySQL)
* Web Browser (Chrome, Edge, etc.)
* PHP 7.x or above

---

Project Structure

* `/Cs619_prototype_PRMS` → Main project folder
* `/database` → Contains SQL file
* `/assets` → CSS, JS, images
* `/modules` → Application features
